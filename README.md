# Competitive-Coding

The Competitive coding is to refine the programming skills obtained in class in order to compete on different programming events & also with their classmates. The main aim of this club is to build an environment of competitiveness.

# Our Aim :- 
## ⭐ To Increase coding literacy.
## ⭐ To create logical thinking among engineering students.
## ⭐ To develop the skill employment for the campus placement.
## ⭐ To remove code phobia.
## ⭐ To create incubation environment for skilled developers.
